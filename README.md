# IEC 104/101 规约解析器

双协议解析工具，支持 **IEC 60870-5-104** 与 **DL/T634.5101-2002** 规约的实时解析和日志文件批量解析。

## 功能特性

✨ **双协议支持**
- IEC 104 可变帧格式：`68 L L ... CS`
- IEC 101 双帧式：固定帧 `10 C A1 A2 CS 16`、可变帧 `68 L L 68 C A1 A2 ... CS 16`
- 自动协议识别或手动标注 `[101]` / `[104]`

📊 **完整的ASDU解析**
- 遥测 (YC)：TI=9, 11, 13 等
- 遥信 (YX)：TI=1, 3, 30, 31（含SOE）
- 遥控：TI=45, 46
- 电能量：TI=206, 207
- 参数传输、故障事件、对时、总召等

🗄️ **点表数据库支持**
- 加载SQLite点表文件（`.db`）或JSON格式
- 前端加载（基于sql.js WASM），无需后端处理
- 自适应101为全点表、104为转发表
- 实时显示遥测/遥信点名

📄 **日志文件解析**
- 支持上传log文件逐行解析
- 保留原始调试前缀（Tx/Rx时间戳等）
- 支持输入框粘贴小段log文本快速验证
- 结果下载为带注解的log文件

## 快速开始

### 安装依赖
```bash
npm install
```

### 开发模式（热重载）
```bash
npm run dev
```
访问 http://localhost:33104

### 生产模式
```bash
npm start
```

### 上传到远程服务器
编辑 `.env` 文件设置SSH情况，然后：
```bash
npm run upload
```

## 使用说明

### 实时解析（index.html）
1. **可选**：加载点表数据库（支持`.db`或JSON）
2. 在文本框输入十六进制报文，每行一帧
3. 点击「🔍 解析报文」或按 `Ctrl+Enter`
4. 查看各协议的ASDU解析结果

### 日志文件解析（fileParser.html）
1. **可选**：加载点表数据库
2. 上传`.log` / `.txt`文件，或在输入框粘贴log文本
3. 选择协议模式（推荐自动识别）
4. 点击「▶ 解析 Log」
5. 下载带注解的结果文件

### 环境变量
如果你需要将项目运行再公网服务器上需要修改环境变量
```
SSH_HOST=host
SSH_USER=user
SSH_PASSWORD=Password
SSH_PORT=22
```
上传路径需要在upload脚本里面修改
