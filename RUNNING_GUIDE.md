# IEC 104/101 解析器 - Electron 桌面版

## 快速开始

### 系统要求
- Windows 7 及以上
- 无需安装 Node.js 或其他依赖

### 安装步骤
1. 下载 `IEC104Parser-1.1.0-win.zip`
2. 解压到任意目录 (例如 `C:\Program Files\IEC104Parser\`)
3. 双击运行 `IEC104Parser.exe`

### 首次运行
- 应用会自动启动本地服务器，监听 `http://localhost:33105`
- 系统可能会提示 Windows Defender，选择"允许"
- 浏览器会自动打开 IEC 解析器界面

### 功能
- **ASCII/HEX 解析**：支持 IEC 60870-5-104 (104协议) 和 DL/T634.5101 (101协议)
- **日志分析**：直接上传 log 文件批量解析
- **数据导出**：解析结果可导出为 JSON 格式

### 手动打开
如果浏览器未自动打开，手动访问：
```
http://localhost:33105
```

## 常见问题

**Q: 闪烁一下就关闭了？**
- 检查 33105 端口是否被占用
- 可在环境变量设置 `PORT` 来改变监听端口

**Q: 上传文件提示 413 错误？**
- 当前限制为 10MB，大文件需分割上传

**Q: 需要卸载怎么办？**
- 直接删除解压文件夹即可，无残余文件
